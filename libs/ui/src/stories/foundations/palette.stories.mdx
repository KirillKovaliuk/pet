# Палитра

Дизайн-система использует целенаправленный набор цветовых стилей как идеальную отправную точку
для любого бренда или проекта. Когда речь идет о цвете, контрастность имеет решающее значение для
обеспечения разборчивости текста. Были добавлены коэффициенты контрастности WCAG 2.2 в систему
цветов, чтобы мы могли быть уверены, что разрабатываем дизайн с учетом требований доступности.
